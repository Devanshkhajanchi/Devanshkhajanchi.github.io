Welcome to my space

  • Manuscript

    Typesetting a Seminal Design Text

    A foundational design text was selected from Readings.design, analyzed, and responded to. Both the original text and the response were typeset with a focus on typography, hierarchy, and expressive form. The final execution was coded as a web page, incorporating critical feedback through an iterative design process.

  • Spread

    Collaborative Responsive Typography

    A responsive web layout was designed and implemented for a selected text in collaboration with a partner. Consistent typographic expression was explored across mobile, desktop, and print formats. The project emphasized collaboration, pair programming, and adaptive design principles.

  • Binding

    A Digital Book of Curated Texts

    A cohesive digital "book" was created to collect and connect previous work. A structured reading experience was developed, including a homepage (cover), navigation (table of contents), and individual pages (chapters). The design was made fully responsive, ensuring accessibility across devices and print formats.

  • Links

    Interactive Collection Interface

    A themed collection was assembled using Are.na as a CMS, and a custom web interface was designed and built using its API. The collection was structured to highlight relationships between items, and JavaScript functionality was integrated to allow for interactive exploration.

  • Functions

    Capstone Web Project

    A real-world problem was identified, researched, and addressed through a web-based solution using HTML, CSS, and JavaScript. Concept development and iterative implementation were carried out to ensure a functional digital experience. The final result was a fully developed web application designed to meet a specific user need.
